# nand2tetris in golang

A golang simulator for the Hack machine described in The Elements of Computing Systems, by Noam Nisan and Shimon Schocken.

## Subfolders

### LISP

A hardware hack on the Hack machine to support lisp instructions at the hardware level. Described in more detail in https://deosjr.github.io/nand2lisp.html

### SAP

From the book Digital Computer Electronics by Albert Paul Malvino, a different set of architectures simulated in a similar way.
This is also the basis for Ben Eater's breadboard computer in his youtube series.
